# History

3,745 commits. 13 years of fake history. One very green contribution graph.

This repo exists to paint a GitHub contribution heatmap from 2013 to 2026. Every commit is backdated, and none of the code was written by a human with deadlines.

## How It Works

A script generates thousands of commits with randomized dates and messages that chronicle the mass delusion of software engineering across the decades. The commits are pushed incrementally so GitHub's contribution indexer actually counts them.
